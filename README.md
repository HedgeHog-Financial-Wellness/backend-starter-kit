# BACKEND STARTER KIT

The backend starter kit contains all the module for building the scalable rest  api.

## Tech Stack

* HTTP Framework: Hono
  * Hono Zod Api Wrapper
* Logger: Pino, pino-pretty
* Lint: Eslint, antfu eslint
* dev api test: vscode rest client
* scalar
